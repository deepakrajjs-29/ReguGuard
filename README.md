## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge, Safari)
- A Gemini API key — [Get one here](https://ai.google.dev)
- A Firebase project — [Set up here](https://console.firebase.google.com)

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/your-username/reguguard-ai.git

# 2. Navigate into the project
cd reguguard-ai

# 3. Add your API keys to config.js
#    GEMINI_API_KEY=your_key_here
#    FIREBASE_CONFIG=your_config_here
```

### ▶️ Run Locally

```bash
# Simply open in your browser
open index.html
```

> No build tools or package manager required — runs entirely in the browser.
